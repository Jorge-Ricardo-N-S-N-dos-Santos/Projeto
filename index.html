Objetivo do projeto:


Realizar uma pesquisa/questionário, para traçar o perfil dos usuários de postos de gasolina, identificando tanto o consumidor de uma forma geral, quanto os trabalhadores por meio de aplicativos.



Tecnologias Utilizadas:


  • Google Forms: Questionário
  • GitHub: Versionamento e colaboração compartilhada e estrutura do Kanban.
  • Excel: Análise de dados e dashboard
  • Visual Studio Code: HTML e CSS: Criação do link de apresentação
